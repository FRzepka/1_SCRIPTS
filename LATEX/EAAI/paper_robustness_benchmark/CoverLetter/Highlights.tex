\section*{Highlights}
\begin{itemize}
    \item Measurement-only robustness benchmark for embedded battery state estimation.
    \item Direct comparison of CC, CC+SOH, ECM, and data-driven SOC+SOH models.
    \item Benchmark focuses on failure modes, drift, stability, and recovery instead of only clean-data accuracy.
    \item Scenario design covers sensor bias, noise, spikes, missing samples, burst gaps, and timing corruption.
    \item Result package is structured to support a decision matrix for estimator selection under BMS constraints.
\end{itemize}
