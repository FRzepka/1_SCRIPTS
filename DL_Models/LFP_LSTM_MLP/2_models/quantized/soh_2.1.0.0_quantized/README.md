# SOH Quantized Model (v2.1.0.0)

**Active quantized model for State of Health (SOH) prediction.**

## Source Model
- **Checkpoint**: `DL_Models/LFP_LSTM_MLP/2_models/base/2.1.0.0_soh_epoch0120_rmse0.03359.pt`
- **Version**: 2.1.0.0
- **Target**: SOH (State of Health)
- **Architecture**: 128 hidden LSTM + 128 hidden MLP (FP32)

## Quantization Details
- **Method**: Manual per-channel INT8 quantization with INT16 recurrent
- **LSTM W_ih**: INT8 (per-row scales)
- **LSTM W_hh**: INT16 (per-row scales, improved precision)
- **LSTM Activations**: FP32
- **MLP**: Full FP32 (no quantization)
- **Scale Method**: Percentile p99.5
- **Memory Savings**: ~70% vs FP32 baseline

## Features (6)
1. `Testtime[s]`
2. `Voltage[V]`
3. `Current[A]`
4. `Temperature[°C]`
5. `EFC` (Equivalent Full Cycles)
6. `Q_c` (Charge capacity)

## Post-Processing
- **PIN calibration**: Start-one, scale anchor, before_filter
- **Strict filtering**: REL_CAP=1e-4, ABS_CAP=2e-6, 5 passes
- **EMA smoothing**: alpha=2e-4

## Files
- `quant_state_soh_int16hh_p99_5.npz` - Quantized weight pack (NumPy format)
- `quant_meta.json` - Quantization metadata and source checkpoint
- `soh_quant_infer.py` - Standalone Python validation script
- `export_to_c.py` - C header export script
- `c_export/` - Generated C headers for STM32 firmware

## Performance
- **MAE**: 0.000476 (0.048%) - **Better than FP32 baseline!**
- **RMSE**: 0.000638 (0.064%)
- **MAX Error**: 0.00140 (0.14%)
- **Sample Rate**: ~28 samples/s on STM32H753ZI
- **Memory**: ~30% of FP32 LSTM weights
